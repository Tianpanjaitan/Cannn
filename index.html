Cyber Security • Christian P.

About Me

Turning Ideas Into
Digital Security

Perkenalkan, saya Christian Panjaitan — siswa SMK Negeri 9 Medan jurusan Teknik Komputer & Jaringan (TKJ) dengan fokus mendalam di bidang cyber security.

Ketertarikan saya pada keamanan digital berawal saat mengikuti LKS Cyber Security Kota Medan 2026, di mana saya berhasil meraih Juara 2. Pengalaman ini membuka mata saya tentang pentingnya keamanan siber di era digital dan memacu saya untuk terus belajar dan berkembang.

Saat ini saya aktif mempelajari administrasi server menggunakan Debian Linux, pemrograman Python untuk otomasi keamanan, serta mendalami jaringan melalui persiapan Olimpiade Mikrotik. Saya juga pernah berpartisipasi dalam Pameran Gebyar & Expo Pendidikan 2026 menampilkan instalasi jaringan fiber optik FTTH.

Target saya ke depan adalah melanjutkan studi ke jenjang S1 di bidang Teknik Informatika atau Cyber Security pada tahun 2027, serta mendapatkan sertifikasi profesional seperti Mikrotik MTCNA dan CompTIA Security+.

My Journey

Experience & Achievements

Perjalanan, kompetisi, dan pencapaian yang membentuk karir saya di dunia cyber security.

2026 — Kota Medan
Juara 2 LKS Cyber Security Kota Medan

Saya bersyukur dapat mengikuti LKS Cyber Security 2026 di Kota Medan dan meraih Juara 2. Kompetisi ini menguji kemampuan saya dalam berbagai aspek keamanan siber, mulai dari penetration testing, forensik digital, hingga network security. Pengalaman ini tidak hanya memperkuat pengetahuan teknis saya, tetapi juga meningkatkan kemampuan problem solving dan kerja tim dalam situasi kompetitif. Menjadi motivasi besar untuk terus berkembang dan meraih prestasi lebih tinggi di masa depan.

Cyber Security Penetration Testing Forensik Digital Network Security
2025 — Present
Olimpiade Jaringan Mikrotik

Selesai mengikuti olimpiade mikrotik yang diselenggarakan oleh CITRA WEB dengan materi routing, firewall, dan troubleshooting jaringan menggunakan Mikrotik RouterOS. Pengalaman berharga dalam memahami konsep jaringan secara mendalam serta meningkatkan kemampuan praktis dalam konfigurasi dan manajemen perangkat jaringan.

Mikrotik RouterOS Routing Protocol Firewall Troubleshooting
2026
Gebyar & Expo Pendidikan — FTTH Installation

Berpartisipasi dalam pameran pendidikan sebagai perwakilan SMKN 9 Medan dengan menampilkan instalasi jaringan fiber optik FTTH. Bertanggung jawab dalam proses splicing kabel fiber optik, konfigurasi perangkat jaringan, serta memberikan penjelasan kepada pengunjung tentang teknologi FTTH.

Fiber Optik FTTH Splicing Public Speaking Teamwork
2025 — Present
Self-Learning: Python & Linux Server

Mempelajari secara mandiri pemrograman Python untuk otomasi tugas-tugas keamanan siber, serta administrasi server menggunakan Debian Linux termasuk konfigurasi layanan, hardening sistem, dan manajemen pengguna.

Python Linux Debian Server Admin Automation
2026 — 2027
Target & Future Plans

Setelah lulus SMK pada tahun 2026, berencana melanjutkan studi S1 di bidang Teknik Informatika atau Cyber Security. Target mendapatkan sertifikasi Mikrotik MTCNA dan CompTIA Security+, serta berpartisipasi dalam kompetisi CTF.

CTF Competition MTCNA CompTIA Security+ University S1
